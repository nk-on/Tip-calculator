## Tip Calculator App

This is a simple tip calculator app built with HTML, CSS, and JavaScript. It allows you to calculate the tip amount for a bill based on a chosen percentage, the number of people splitting the bill, and the total bill amount.
Features

    Calculate tip amount based on a percentage.
    Calculate tip amount per person.
    Calculate total bill amount including tip.

## How to Use

    Open the index.html file in your web browser.
    Enter the total bill amount in the "Bill Amount" field.
    Choose your desired tip percentage from the radio buttons.
    Enter the number of people splitting the bill in the "Number of People" field.
    Click the "Calculate" button.

## The app will display the following information:

    Tip amount per person
    Total tip amount
    Total bill amount (including tip)

## Dependencies

This app does not require any external libraries or frameworks.
Running the App

    Clone this repository.
    Open the index.html file in your web browser.
